① 执行摘要 — 5 个核心结论（2026年6月）

基于SSBS评分 + 时序信号 + 产业链分析综合判断
🔴 结论1：电力/变压器是最真实的物理瓶颈，被市场低估
变压器交货期已超过2年，是所有DC扩张项目的首要限制。Eaton/GE Vernova订单积压 分别达0.52×和0.95×营收，远超正常水平。芯片侧产能在扩，但电力配送侧积压更难解决。 这是整条产业链中唯一纯物理约束（不能靠钱加速）。
投资含义：ETN、GEV 是被低估的瓶颈受益股 → 买入
🔴 结论2：Hanmi半导体是最有数据支撑的纯买入标的
积压比率1.86×（全链最高），TC粘合剂工序无可替代。已验证领先SK Hynix HBM出货1季度。 SSBS得分全链第一（58.8），且有真实财务数据支撑。HBM3E/HBM4需求至少持续到2027Q2。
投资含义：042700.KS (Hanmi) → 强买入，最高确定性
🔵 结论3：NVDA→SK Hynix→Hanmi 的时序链已验证，当前处于上行段
NVDA B300超强出货（2025Q3-Q4） → SK Hynix HBM3E收入加速（滞后1-2Q，相关性0.93） → Hanmi TC粘合剂收入跟进（再滞后1Q）。该传导链已被历史数据验证，不是假设。 当前NVDA尚未见顶，时序链仍在上行段中段。
投资含义：SK Hynix (000660.KS) 是时序信号最强的标的之一 → 关注→买入
🟢 结论4：DC承包商(STRL/FIX)是信噪比最高的基础设施标的
零地缘风险（纯美国业务），积压覆盖1.1-1.2×营收，领先超大规模商资本支出12-18个月。 市场把它们当建筑股定价（低估值），但它们实际上是AI基础设施的领先指标。 超大规模商将继续签署大额DC项目合同，承包商积压将持续创新高。
投资含义：STRL、FIX → 买入，估值低+信号强+无地缘风险
🟡 结论5：ODM服务器厂（鸿海/广达）是全链最应回避的标的
SSBS评分最低（37分，唯一AVOID节点），定价权评分仅5.7%（全链倒数第一），台湾+中国生产集中度高， 地缘风险0.48。AI繁荣对ODM的传导完全被竞争压缩掉了——他们做的越多利润越薄。 不要因为"AI供应链"概念就买ODM。AI繁荣的利润流向IP/设备/材料端，而不是组装端。
投资含义：2317.TW (鸿海)、2382.TW (广达) → 回避。利润不在这里。
